# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-06

### Added
- Browser-based image compression for JPEG, PNG, WebP, and AVIF formats
- Drag & drop, file browse, and clipboard paste (Ctrl+V) upload methods
- Batch compression with individual and ZIP download options
- Side-by-side live preview comparing original and compressed images
- Quality slider (10-100%) with preset buttons (Low, Balanced, High, Maximum)
- Resize controls with aspect ratio lock, exact dimensions, and percentage scaling
- Format conversion between JPEG, PNG, WebP, and AVIF with transparency preservation
- Dark mode and light mode with system preference detection and persistence
- Keyboard navigation, focus states, and screen reader accessibility
- Toast notification system for user feedback
- Privacy-first architecture with zero server dependencies
- Responsive design for desktop and mobile
- TypeScript with strict mode, interfaces, and modular architecture
